# sendex-contacts
Full CRUD React application to store your contacts. Uses React, Node.js, Express.js, MongoDB


# More Info
install

```
npm install
npm client-install
```

Running the server
```
npm run dev  -  Express & React :3000 & :5000
npm run server  -  Express :5000
npm run client  -  React :3000
```
